# 🧮 Calculadora Simples - Desafio de Programação

✨ **Descrição**  
Crie uma calculadora interativa utilizando Python. O objetivo é praticar a entrada de dados, o uso de operadores matemáticos e a lógica condicional. O programa solicitará ao usuário dois números e o operador matemático desejado, e então exibirá o resultado da operação.

📆 **Requisitos**  
O programa deve:

- Solicitar ao usuário o primeiro número.
- Solicitar ao usuário o segundo número.
- Solicitar o operador matemático desejado (`+`, `-`, `*`, `/`).
- Realizar a operação indicada e exibir o resultado ao usuário.

⚠️ **Restrições**  
❌ O código **deve quebrar** caso o usuário informe números negativos ou entradas não numéricas (como strings).  
❗️ Não é necessário implementar tratamento de erros ou validações.  
🔢 Apenas operadores básicos devem ser suportados.

💪 **Desafio Extra**  
Se quiser ir além, tente implementar uma nova versão com validações:

- Impedir que o usuário informe valores inválidos (strings ou negativos).
- Exibir mensagens de erro amigáveis.

🌟 **Exemplo de Execução**
```

Digite o primeiro número: 10
Digite o segundo número: 5
Digite o operador (+, -, \*, /): \*
Resultado: 50

```

Agora é com você! Mãos à obra e bons cálculos! 🚀
